Formatação de Texto

Introdução

Anteriormente foi possivel definir e compreender os níveis de titulos de h1-h6,
que fornece a possibilidade de trabalhar com titulos e subtitulos.
Também utilizamos elementos para tornar o conteudo negrito, italico, subscrito e sobrescrito. Para organizar, também é possível utilizar quebras de linhas - para pular uma linha - ou uma quebra de linha horizontal,
que insere uma nova linha horizontal na página.


Utilizamos o elemento strong que permite enfatizar conteudios que tenham grande importância,
assim como o elemento em, que permite enfatizar uma palavra ou expressão.
Para lidar com citações, utilizamos a tag blockquote, q, e cite. Cada uma com suas respectivas caracteristicas,
por exemplo, vamos definir algumas citações:

O mundo não se divide em pessoas boas e más. Todos temos luz e trevas dentro de nós. O que importa é o lado o qual decidimos agir. Isso é o que realmente somos!

Sofrer é tão humano quanto respirar. Harry Potter

Caso seja necessário utilizar abreviações, usamos a tag abbr.

Se desejar inserir nome ou endereço do autor - ou uma entidade - da página, podemos utilizar o elemento address.

E-mail: anakin@starwaras.com

Anakin, Tatooine

É possível manter alterações textuais, alterando apenas a formatação:

Utilizando o elemento del para um texto antigo: Texto anterior
Utilizando o elemento ins para um novo texto: Novo texto
Também é possível ser enfatizando o que não é mais relevante, porém, não deve ser removido:
Utilizando a tag s: Esse conteúdo não é mais relevante, mas deve permancer na página.
