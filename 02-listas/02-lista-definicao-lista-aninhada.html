Listas de Definições são declaradas utilizando pares de elementos, para o termo e para a descrição do mesmo.


Strong
O elemento strong é utilizado para enfatizar um texto de grande importância.
Em
A tag "em" enfatiza uma palavra ou expressão estrangeira.

É possível criar listas com sublistas utilizando o elemento ul ou ol
